# AppAutoTest
一个基于appium的APP POM自动化测试
目录从上到下依次为 基类，公共类，配置，页面，结果，测试用例管理， 测试数据， 实现了 老东家瓜子二手车 App 部分功能测试，在unittest运行框架测试并输出html报告， 其中log数据可实时输出控制台也可写入result文件夹下log文件中保存
